# Matriz MoSCoW — Atributos de Calidad

## MUST HAVE

| ID      | Atributo                          | Justificación |
|--------|----------------------------------|--------------|
| RNF-001 | Disponibilidad 24/7 (≥99%) | Sistema crítico para monitoreo en tiempo real |
| RNF-005 | Cifrado de contraseñas | Requisito mínimo de seguridad |
| RNF-007 | Conexión segura (HTTPS/TLS) | Protección de datos IoT |
| RNF-003 | Tiempo de respuesta ≤ 15s | Alertas deben ser oportunas |
